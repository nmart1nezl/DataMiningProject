# Data Mining Project

This repository contains the deliverables for a university Data Mining project. It brings together the raw and processed datasets, exploratory analysis notebooks, classification notebooks, association-rule mining work, clustering analysis, and the written and presentation materials produced throughout the course milestones.

## Project Overview

The goal of the project is to apply the main data mining pipeline end to end: data preparation, exploratory analysis, feature engineering, supervised learning, unsupervised learning, and association analysis. The repository is organized so each stage of the work can be reviewed independently while still reflecting the full workflow used in the course.

## Repository Structure

- datasets/ - Source, cleaned, and task-specific datasets used across the project.
- notebooks/ - Jupyter notebooks for preprocessing, exploratory analysis, association rules, clustering, and classification.
- articles/ - Final written reports and milestone papers.
- slides/ - Presentation decks for the project milestones and final defense.

## Datasets

The project includes the following datasets:

- Billionaires Statistics Dataset - Original.csv - Original dataset used as the starting point for the analysis.
- dataset_cleaned.csv - Cleaned version of the main dataset after preprocessing.
- classifiers_dataset.csv - Dataset prepared for classification tasks.
- clustering_dataset.csv - Dataset prepared for clustering analysis.
- transactional_dataset - association.csv - Transactional format dataset used for association rule mining.

## Notebooks

Main notebooks available in the repository:

- Preprocessing_and_Data_Analysis_Notebook.ipynb - Data cleaning, preprocessing, and exploratory analysis.
- Association_Notebook_Project_Milestone_2.ipynb - Association rule mining workflow and results.
- Clustering_Notebook_Project_Milestone_2.ipynb - Clustering analysis and interpretation.
